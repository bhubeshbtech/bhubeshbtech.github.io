WoundChron

            
Final-Year Project · Wet-Lab + Computational + Software

          

          2025 – 2026
        

        
Model-based estimation of wound aging using plant-derived nanoparticles and bioinformatics tools — a wound-aging platform built on green-synthesized nanoparticles, validated computationally, and shipped as a working web application.

        

          ZnO Nanoparticle Synthesis
          Molecular Docking
          GEO Transcriptomics
          React.js
        

        
    
          
  • Green-synthesized ZnO nanoparticles from Ageratum conyzoides extract, characterized via UV-Vis, XRD, and FTIR spectroscopy
    • 
          
  • Performed molecular docking (quercetin, −8.3 kcal/mol vs. target 3MME) and GEO transcriptomic biomarker analysis (Scanpy) to support the wound-aging model
    • 
          
  • Engineered a JavaScript-based image-analysis pipeline (Gaussian blur, Otsu thresholding, healing-rate estimation) inside a deployed React.js web app
    • 
          
  • Team project with Balaji L and Lokesh M, under Dr. K.R. Preethy, Dept. of Biotechnology
    •
