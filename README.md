- **[WoundChron](https://bhubeshbtech.github.io/WoundChron/)** — model-based estimation of wound aging using plant-derived nanoparticles and bioinformatics tools: green-synthesized nanoparticle characterization, molecular docking, GEO transcriptomic analysis, and a deployed React.js app. Final-year project with Balaji L and Lokesh M, under Dr. K.R. Preethy.
